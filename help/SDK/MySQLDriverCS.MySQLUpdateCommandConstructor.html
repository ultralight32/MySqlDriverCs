Parameters

conn
Database connection
FieldsAndValues
Pairs of Fields and values as an object array. Examples: 
new object[,]{{"SettingID",SettingID},{"SettingValue",Value}}

new object[,]{{"SettingID","times"},{"SettingValue",100}}

null
Table
Table name
WhereParamFields
Tripplets of parameters, operand and its values to match (null - nothing). For example: 
new object[,]{{"myfield","=",myvalue},{"myfield2","<>",myvalue2}}
 
new object[,]{{"myfield","=",100}}
 
new object[,]{{"myfield","=",myvalue}}

null
WhereNoparamFields
Tripplets of parameters, operand and expressions (or other fields for joining) to match (null - nothing). For example: 
new object[,]{{"myfield","=","myfield2"}}

null

Example

Example updating SettingValue by SettingID in Settings table

using MySQLDriverCS;
...
MySQLConnection DBConn;
DBConn = new MySQLConnection( new MySQLConnectionString("myhost","mydatabase","mylogin","mypassword").AsString );
DBConn.Open();
...
new MySQLUpdateCommand(DBConn,
    new object[,] {{"SettingValue",Value}},
    "Settings",
    new object[,] {{"SettingID","=",SettingID}},
    null
);
...
DBConn.Close();
