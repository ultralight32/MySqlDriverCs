Return Value

A Int32 containing a value that reflects the sort order of a as compared to b. The following lines defines the conditions under which the returned value is a negative number, zero, or a positive number. Any negative number if a.ParameterName.Length < b.ParameterName.Length. Zero if a.ParameterName.Length == b.ParameterName.Length. Any positive number a.ParameterName.Length > b.ParameterName.Length.
